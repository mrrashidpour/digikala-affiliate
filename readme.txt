== Installation ==

to install the plugin use these steps.

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click on digikala affiliate link on the rightside menu to configure the plugin.
4. Enter your Links made from the page in the affilio admin panel (https://panel.affilio.ir)
5. Save the settings 
6. All the links of digital goods in the posts will be automatically changed to your link
